# Tp0t OJ

Tp0t战队开发的CTF用OJ平台。

> 当前分支为前端开发分支，请勿开发其他内容
> 
> 第一次部署项目，请注意运行`npm install`

## 开发指南

### 前端

+ **请在`app`目录下打开vscode**
+ **npm相关命令，请在`app`目录下运行**
+ 总之请将`app`做为工作目录

### 后端

+ **IntelliJ IDEA请在`server`目录下打开**
+ `server`目录应视同工作目录

1. 数据库定期扣分回滚点设计

### 接口相关

+ GraphQL的schema文件定义在`server/src/resources/schema`目录下
